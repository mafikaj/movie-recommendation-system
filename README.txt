# Semantic Similarity Analysis

This project uses spaCy and cosine similarity to recommend movies based on the similarity of their descriptions. The movie descriptions are provided in the `watch_next.py` script.

## How to Use

1. Make sure you have Python installed on your machine.

2. Install the required dependencies by running the following command:

3. Run the `watch_next.py` script:

4. Enter a movie description when prompted, and the script will recommend the most similar movie.

## Files

- **watch_next.py:** The main script for recommending movies.
- **Dockerfile:** Docker configuration file for containerizing the application.
- **requirements.txt:** Lists the required Python packages and their versions.
- **semantic_similarity.txt:** Contains the link to the remote Git repository.
